# NagataFactoriality

A Lean 4 / Mathlib project aimed at a **publication-ready formalization of Nagata's factoriality theorem**.

## Draft abstract

We formalize in Lean 4 / Mathlib a prime-generated version of Nagata's factoriality theorem for noetherian integral domains. The development isolates the localization lemmas needed to transfer irreducibility and primality between a domain and a localization at a submonoid generated by prime elements, then packages these ingredients into a reusable Nagata theorem. As a concrete application, we derive a polynomial unique factorization result from a Laurent-polynomial localization hypothesis, showing that the formalized theorem does genuine mathematical work beyond thin wrappers around existing library instances.
