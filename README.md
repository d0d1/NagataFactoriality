# NagataFactoriality

A Lean 4 formalization of **Nagata's Factoriality Theorem**: if $R$ is a Noetherian integral domain and $S \subseteq R$ is a multiplicative set generated by prime elements such that the localization $S^{-1}R$ is a UFD, then $R$ itself is a UFD.

## Roadmap

### Phase 1: Foundations
- [ ] Integral domains, units, associates, irreducibles, primes
- [ ] Divisibility theory and GCD domains
- [ ] Noetherian rings and ACC on ideals
- [ ] UFDs: definition, characterization via irreducible = prime

### Phase 2: Localization
- [ ] Multiplicative subsets and localization $S^{-1}R$
- [ ] Universal property of localization
- [ ] Prime ideals under localization (going-up, going-down)
- [ ] Factorization in localized rings

### Phase 3: Key Lemmas
- [ ] If $p \in R$ is prime and $p \notin S$, then $p$ remains prime in $S^{-1}R$
- [ ] Lifting factorizations from $S^{-1}R$ back to $R$
- [ ] Nagata's lemma: irreducibles in $R$ not in $S$ are prime

### Phase 4: The Main Theorem
- [ ] **Nagata's Theorem**: $R$ Noetherian, $S$ generated by primes, $S^{-1}R$ UFD $\Rightarrow$ $R$ UFD
- [ ] Corollaries: $R$ UFD iff $R[x]$ UFD (via Gauss's lemma)

### Phase 5: Applications
- [ ] Polynomial rings over UFDs
- [ ] Gauss's lemma as a consequence
- [ ] Laurent polynomial rings
